  • New Dashboard for better user experience.

v4.1.1
March 2026
Major

Flagship release bringing Sigma rule evaluation directly into the collection pipeline, plus BitLocker key harvesting, expanded IOC coverage, and a raft of reliability fixes across all three platforms.

✦ New
  • Sigma rule support.
  • Malware hash updates (abuse.ch).
  • Script execution logging.
  • BitLocker key collection.
  • Added malicious executable check for Linux.
  • Added malicious shell commands check for Linux.
▲ Improved
  • Encryption/Decryption.
  • Browser history & IOC detection.
  • Updated to modern commands.
✔ Fixed
  • Fixed some carriage return errors in Linux.
  • Fixed report title encoding issue when -TITLE contains special characters.
  • Fixed -WEBLOGS path detection failing on non-default IIS log locations.
  • Linux: fixed false-positive malicious command detections triggered by legitimate package manager strings.

Windows-only PowerShell toolkit. Monolithic script collecting system info, user accounts, network state, processes, startup persistence, and scheduled tasks with plain-text output.

  • Added Ability to grab the browsing history of all users.
