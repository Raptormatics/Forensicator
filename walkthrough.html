// Getting Started

Prerequisites

  • PowerShell 5.1 or later
  • Administrator Privileges
  • Execution Policy set to Bypass (-ExecutionPolicy Bypass)
⚠️
Run as Admin is mandatory for Event Log access and Memory acquisition.
  • Bash 4.0+
  • Root access (Sudo)
  • Native OS tools (Forensicator has zero dependencies)
// Deployment

Installation

Forensicator is a portable toolkit. No installation is required.

Bash / PowerShell
git clone https://github.com/Johnng007/Live-Forensicator.git
// Execution

Windows — Basic Run

PowerShell
# Simple collection
.\Forensicator.ps1

# Deep dive with RAM and Logs
.\Forensicator.ps1 -EVTX EVTX -RAM RAM -OPERATOR "John Doe"
